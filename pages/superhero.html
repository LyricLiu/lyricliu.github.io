Superheroes Around theWorld

Twitter data mining and visualization.

Personal Work

10/2016

Introduction

Superheroes around the World is a data visualization online webpage which aims to visualize stream twitter data about superheroes including Super Man, Spider Man, Black Widow, Captain America, Bat Man, Wonder Woman, Thor and Iron Man. This project mines the tweets geolocations of each hero and visualizes them on a 3D earth model. It also displays the number of tweets that has received and the contents of tweets. Based on the character settings, each hero is represented by one color. Each piece of data displayed is color coded.

This project is more like a tool for those who want to analysis superhero data.
